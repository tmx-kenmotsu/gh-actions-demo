## ここは

GitHub Actionsのスターターパッケージのデモ環境です。

## 利用ツール

* ビルド：Apache Maven
* 静的解析：SpotBugs
* 単体テスト：JUnit
* 成果物の保存：GitHub Actions Artifacts
* 通知：Microsoft Teams
